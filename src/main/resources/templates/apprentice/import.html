Importer des apprentis depuis Excel

Instructions pour l'import

Le fichier Excel doit contenir les colonnes suivantes (respecter exactement ces noms) :

Colonnes obligatoires :

  • nom - Nom de l'apprenti
  • prenom - Prénom de l'apprenti
  • email - Adresse email (doit être unique)
  • entreprise - Raison sociale de l'entreprise (doit exister)
  • maitre_apprentissage - Nom complet du maître d'apprentissage (doit exister)
  • tuteur_enseignant - Nom complet du tuteur enseignant (doit exister)

Colonnes optionnelles :

  • telephone - Numéro de téléphone
  • programme - Programme d'études
  • majeure - Majeure
  • niveau - Niveau (par défaut : I1)
  • annee_academique - Année académique (par défaut : année en cours)
  • mission_mots_cles - Mots-clés de la mission
  • mission_metier_cible - Métier cible de la mission
  • mission_commentaires - Commentaires sur la mission
  • remarques_generales - Remarques générales

Notes importantes :

  • Le fichier doit être au format Excel (.xlsx)
  • La première ligne doit contenir les en-têtes
  • Les entreprises, maîtres d'apprentissage et tuteurs doivent déjà exister dans le système
  • Les emails doivent être uniques
